JavaDoc 常用标签

JavaDoc 是 Java 中自带的文档工具，用于生成 Java 代码的 API 文档。在 JavaDoc 注释中，可以使用各种标签来标记不同的注释信息，以下是一些常用的 JavaDoc 标签：

@param

用于描述方法参数的说明。

@return

用于描述方法返回值的说明。

@throws

用于描述方法抛出的异常的说明。

@deprecated

用于标记不推荐使用的方法或类。

@see

用于指向相关的其他类、方法或文档。

@link

用于指向相关的其他类、方法或文档，并提供一个可点击的链接。

@since

用于描述方法或类添加的版本信息。

@version

用于描述方法或类的版本信息。

@author

用于描述方法或类的作者信息。
